Transiciones y Animaciones con CSS

Transiciones con CSS

Permite cambiar suavemente una propiedad CSS de un estado a otro durante un período de tiempo.

PRopiedades de transiciones

  • transition-preoperty: Define qué propiedad cambiar
  • transion-duration: Define la duración del cambio (s o ms)
  • tansition-timing-function: Controla la aceleración o desaceleración
    • linear -> Velocidad constante.
    • ease (por defecto) -> Comienza lenta, acelera y luego desacelera.
    • ease-in -> Comienza lenta, acelera
    • ease-out -> Comienza rapida y desacelera.
    • ease-in-out -> Comienza y termina lenta.
  • transition-delay: Retrasa el inicio de la transición
